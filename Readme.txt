Practica FINAL
Brazo robot de 4 ejes de libertad

En esta practica se muestraun brazo robot que intenta
emular el funcionamiento de un brazo robot industrial.
Mismo que realiza su rutina ordinariamente y vía pc
se puede controlar cada motor del mismo, ademas,
cuenta con un boton de paro de emergencia.

Para esta practica se utilizo:

5 servos
1 chasis para el brazo
1 Arduino UNO
1 Protoboard
1 Relevador
1Laptop
1 Base de madera para instalar el brazo
Cables Dupont
1 Eliminador de 12v y 2A
1 push buton
1 led
2 ressitencias de 220 ohms
